## Project Overview

**Readly** is an immersive AI reading companion. It extracts high-dimensional insights from long-form articles, helping users turn reading into structured knowledge.

**Core Value:** Turning fragmented reading into a structured knowledge loop, reducing the cognitive burden of processing long texts while retaining the original context.

## 01 Problem

Some AI translation and summarization tools feel "robotic": stiff machine translations, rough paragraph summaries, or AI hallucinations disconnected from the original text. For high-value long-form content, readers don't need more noise; they need high-dimensional insights that cut through the surface.

## 02 Insight: Reading Should End with Precipitation

The real value of reading long-form articles lies in the "Aha!" moments. Instead of giving users a wall of summarized text, we should anchor insights to their original context and allow users to add their own takeaways simultaneously.

## 03. One Card = One Insight

Instead of traditional summarization, Readly adopts a "One Card = One Insight" philosophy. It breaks down long articles into 4-6 core cognitive anchors. Each card contains a sharp insight and a "定位到原文" (Trace back) button that smoothly scrolls to the exact original sentence.

## 04. Experience Design: Why Side Panel?

- **Side-by-Side Context:** Users can read the original article while viewing the insights and takeaways, maintaining chronological order (Document Order).
- **Smart Validation:** Uses an "AI Judge" mechanism to softly intercept non-article pages with an elegant prompt.

## 05. Next Steps

- **Knowledge Precipitation:** Implement one-click synchronization to GitHub Pages, turning the reading companion into a true knowledge loop engine.
- **Local Markdown Export:** Support downloading insights and takeaways as a clean Markdown file.
